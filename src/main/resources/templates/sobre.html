Sobre o Sistema

Sistema de Gerenciamento para Clínica Veterinária

Visão Geral

Este sistema foi desenvolvido como projeto acadêmico para demonstrar a implementação de 9 Padrões de Projeto GoF (Gang of Four) em conjunto com a Arquitetura MVC usando Spring Boot.

O sistema oferece funcionalidades completas para gerenciamento de uma clínica veterinária, incluindo cadastro de pets, agendamento de consultas, sistema de pagamentos e notificações.

Padrões de Projeto Implementados

Builder

Construção de objetos complexos como pets com validações.

Factory

Criação de objetos complexos como pets e consultas de forma padronizada.

Singleton

Garante uma única instância de classes importantes como conexões de banco.

Facade

Interface simplificada para operações complexas da clínica.

Adapter

Integração com sistemas externos de pagamento.

DAO

Acesso a dados padronizado para pets e atendimentos.

Strategy

Diferentes estratégias de cálculo de preços para consultas.

Observer

Sistema de notificações para veterinários e clientes.

Command

Agendamento e reagendamento de consultas com histórico.

Arquitetura MVC

Model

Entidades e regras de negócio

View

Interface web com Thymeleaf

Controller

Controle de fluxo e validações
